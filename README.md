# Présentation du projet

Ce projet est un site internet dédié à la chatterie Chats-Shibala. Il se compose d'une page d'accueil présentant la chatterie et d'une page de présentation de la race de chat élevée : le Maine Coon. Une page « À propos » expliquant mon choix de ce sujet

## Technologies utilisées

Ce projet utilise du HTML/CSS avec Flexbox pour assurer le design responsive.
